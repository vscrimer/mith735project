My TEI document

Because algorithmic parsing utilities like AnyStyle struggle to accurately locate and parse citations that are included in discursive foot/end notes which happens to be the primary mode of listing citations in these journals, one possible iteration of this project could be building a public database of reference notes. This could be a crowd sourced project in which we make available an online library of .txt files containing only the bibliographic notes from a single issue of a particular journal. Users could then encode those files using a shared markup scheme like TEI. The files could then be parsed or searched to generate data for modeling.
